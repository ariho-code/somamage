## Parent Access (NO LOGIN REQUIRED)

Parents **DO NOT** have regular login accounts. Instead:

1. **Access Method:** Use the "Parent Login" tab on the login page
2. **Authentication:** Enter the **Student Admission Number** (auto-generated by the system)
3. **Auto-created:** Parent accounts are automatically created when they first login using a student's admission number
4. **Password:** Initially set as `parent{ADMISSION_NUMBER}` (e.g., `parent20250001`)

### How Parents Login:
1. Go to login page
2. Click "Parent Login" tab
3. Enter student admission number (e.g., `20250001`)
4. System automatically creates parent account if not exists
5. Parent can view:
   - Student report cards
   - Student fee balances
   - Student academic performance

## Student Access

**IMPORTANT:** Students **CANNOT** login to the system. They do NOT have user accounts.

- Parents access student information using the student's admission number
- Student admission numbers are auto-generated when students are added
- Format: `YYYYNNNN` (e.g., `20250001`, `20250002`, etc.)

## Adding New Users

### To add more users:
1. **Admin users:** Use Django admin panel at `/admin/`
2. **Teachers/Staff:** Use the "Add Staff" option in the system
3. **Students:** Use "Add Student" - admission numbers are auto-generated
4. **Parents:** Automatically created when they login with student admission number

### To clear and reset all users:
```bash
python manage.py setup_users --clear
```

This command will:
- Clear existing parent, bursar, teacher, and headteacher accounts (keeps superadmin)
- Clear all students and guardians
- Create fresh user accounts with the credentials listed above

2. **Admission Numbers:** Auto-generated when students are added. Format: `{YEAR}{4-digit-number}` (e.g., `20250001`)

3. **Parent Accounts:** Created automatically when parent logs in with student admission number. Username is guardian phone number.

4. **Password Security:** Change all default passwords in production!

5. **First Login:** When a parent first logs in with an admission number, a parent account is created with default password. They can request password reset if needed.
